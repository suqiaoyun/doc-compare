# 文档对比助手 (MatchDiff)

文档对比助手是一款基于浏览器前端开发的轻量级内容核对工具。该工具支持多种办公文档格式的解析与差异化展示，旨在解决人工核对文档修订点时效率低、易遗漏的问题。

## 功能特性

* **多格式兼容**：集成多种解析引擎，支持 `.pdf`、`.docx`、`.xlsx`、`.xls` 及 `.txt` 格式。
* **同步滚动比对**：左右双视窗布局，支持原生滚动联动，确保对比位置实时对齐。
* **差异自动标记**：
    * **表格模式**：基于坐标系比对，自动高亮内容不一致的单元格。
    * **文本模式**：基于行差异算法（Diff-line），识别内容的增加与删除。
* **多工作表支持**：针对 Excel 文件，提供 Sheet 切换功能，支持特定工作表的数据核对。
* **交互式差异清单**：侧边栏实时汇总差异点，支持点击清单条目自动定位至文档对应位置。
* **PDF 结构化展现**：自动解析 PDF 页码并在比对视图中插入页码分隔标识。

## 技术规格

本程序采用纯前端架构，无需后端服务器处理，确保了数据的隐私安全性。

* **核心库**：
    * `SheetJS (XLSX)`：处理 Excel 文件的解析与 JSON 转换。
    * `Mammoth.js`：解析 `.docx` 文件并提取原始文本。
    * `PDF.js`：渲染及提取 PDF 文本内容。
    * `jsdiff`：执行文本差异计算算法。
* **运行环境**：支持现代主流浏览器（Chrome, Edge, Firefox, Safari）。

## 使用指南

1.  **文件上传**：通过点击或拖拽方式将“原件”与“修订件”分别载入对应的上传区域。
2.  **配置选项**：若上传文件为 Excel，请在下方的下拉菜单中确认需核对的工作表（Sheet）。
3.  **执行比对**：点击“执行核对”按钮，系统将生成比对视图。
4.  **差异定位**：通过右侧“差异清单”查看汇总，点击条目可快速跳转至差异发生的具体坐标。
5.  **视图控制**：可点击侧边栏关闭按钮隐藏清单以扩大比对视野，通过右下角浮动按钮重新唤起。

## 注意事项

* 本工具的 PDF 解析基于文本层提取，不支持图片扫描件格式的 PDF 比对。
* Excel 比对目前采用单元格文本值对齐逻辑，暂不包含单元格格式（如颜色、字体）的差异核对。
* 大型文档的解析速度取决于用户本地设备的硬件性能。

## 部署说明

本工具为单 HTML 页面结构，无需安装依赖环境。将源代码保存为 `.html` 文件后，直接使用浏览器打开即可运行。
